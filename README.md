# FORO

FORO è una piattaforma SaaS cloud per studi legali italiani, progettata come **Scrivania Digitale a widget**.

## Stato

Questo workspace contiene la baseline documentale di prodotto. Non contiene ancora il codice applicativo.

## MVP

1. Calendario;
2. Documenti;
3. Email;
4. Clienti;
5. Pratiche / Fascicolo.

## Uso

Prima di progettare o implementare una feature:

1. leggere `CLAUDE.md`;
2. seguire `PROJECT_INDEX.md`;
3. verificare `DECISIONS.md`;
4. identificare le business rule;
5. leggere le specifiche frontend/backend/database pertinenti;
6. risolvere le decisioni aperte che bloccano il lavoro.
